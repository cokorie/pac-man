# Pac-Man

### __Pac-Man__
Enjoy this recreation of the classic game, Pac-Man. Move through the game collecting food and escaping ghosts, and see how much you can achieve.

Each food pellet is worth 10 points to your total, and you are allotted 3 lives to get the highest possible score that you can.

Can you eat every pellet, and attain the highest score possible? Go ahead and try your best, and have fun playing the game!

### __Technologies Used__
##### _HTML_
  The HTML code is used to house the: 
  * Game title
  * Game board

##### _CSS_
  The CSS code is used to house the styling of the:
  * Game body
  * Game board

##### _JavaScript_
  The JaveScript code is used to house the functions & logic of:
  * Game board width and height
  * Game board mapping and proper loading
  * Enemy ghost images and movements
  * Pacman images and movements
  * Game food placements
  * Game speed
  * Player and enemy collisions, for restarting positions
  * Game resetting after lives are finished
